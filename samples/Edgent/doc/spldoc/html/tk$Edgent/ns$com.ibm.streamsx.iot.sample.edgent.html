Apache Edgent

Apache Edgent is a programming model and runtime that can be embedded in gateways and devices. An open source solution for implementing and deploying edge analytics on varied data streams and devices.

Edgent interacts with IBM Streams through IBM Watson IoT Platform. Edgent locally separates the interesting from the mundane, so you donâ€™t have to send every sensor reading over a network. If 99% of readings are normal, Edgent detects the 1% anomalies and just sends those as device events for further analysis with IBM Streams.

IBM Streams applications analyze device events from Edgent applications and then can control individual devices by sending device commands based upon the analytics.

Apache Edgent IBM Watson IoT Platform connector

Edgent provides a connector to IBM Watson IoT Platform to allow easy submission of device events and reception of device commands.

Its API is described here: org.apache.edgent.connectors.iotp.IotpDevice

A sample application is provided: org.apache.edgent.samples.connectors.iotp.IotpSensors

Operators

  • IotpSensors: Application that matches the Apache Edgenet Watson IoT Platform sample org.apache.edgent.samples.connectors.iotp.IotpSensors. No analytics are performed, just a warning is echoed back to the device as a device command if a device event with identifier sensor is received.

Types

  • AggInfo: SPL type for aggregation info in device event.
  • SensorInfo: SPL type for aggregation sensor info in device event.
  • WarningInfo: SPL type for device command payload sending a warning message.
